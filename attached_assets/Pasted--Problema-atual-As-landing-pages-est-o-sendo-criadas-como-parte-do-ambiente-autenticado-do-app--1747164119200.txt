## Problema atual:
- As landing pages estão sendo criadas como parte do ambiente autenticado do app
- Isso está causando problemas de CSP e de acesso para usuários não autenticados
- As landing pages devem ser o ponto de entrada do app, sem necessidade de autenticação

## Solução necessária:

1. Criar rotas públicas dedicadas para as landing pages:
   - Modifique o arquivo de rotas para garantir que as landing pages estejam fora do middleware de autenticação
   - As rotas devem ser `/` (página inicial) e `/for-sellers` (para lojistas)

2. Implementar essas landing pages como páginas totalmente públicas:
   ```tsx
   // Estrutura sugerida
   const AppRoutes = () => (
     <Routes>
       {/* Rotas Públicas - Sem necessidade de autenticação */}
       <Route path="/" element={<CustomerLanding />} />
       <Route path="/for-sellers" element={<SellerLanding />} />
       
       {/* Resto do app - Protegido por autenticação */}
       <Route element={<ProtectedRoutes />}>
         <Route path="/dashboard" element={<Dashboard />} />
         {/* Outras rotas protegidas... */}
       </Route>
     </Routes>
   );

Para as imagens nas landing pages:

Criar pasta /public/landing/ com subpastas para cada landing page
Usar caminhos de imagem diretos: <img src="/landing/sellers/hero.jpg" alt="..." />
Sem necessidade de criar componentes especiais, usar HTML/JSX padrão


Implementar os botões de login/cadastro nas landing pages:

Criar botões que abram os modais de login/cadastro
Adicionar na barra de navegação superior e em pontos estratégicos da página
Exemplo: <button onClick={() => setShowLoginModal(true)}>Entrar</button>


Certificar-se que os modais de login/cadastro:

São componentes independentes que podem ser importados nas landing pages
Não requerem autenticação para serem exibidos
Redirecionam para a área autenticada após login/cadastro bem-sucedido



IMPORTANTE: Verifique a estrutura atual do roteamento para entender onde está o middleware de autenticação e garantir que as landing pages fiquem completamente fora dessa proteção.
